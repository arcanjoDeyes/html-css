A importancia da semantica no HTML5

A semântica no HTML5 é fundamental para criar páginas web bem estruturadas, acessíveis e otimizadas para mecanismos de busca. Ela consiste em usar as tags HTML corretas para cada tipo de conteúdo, atribuindo significado e contexto ao código, em vez de depender apenas de tags genéricas como <div> para estilização.
